# AWS-DEPLOY

### Prerequisites

- Node
- NPM

### Installing

At the root directory, run the following commands:

```sh
yarn or npm install
```

```sh
yarn start or npm run start
```

## Built With

- Node.JS
- Typescript
- AWS S3
